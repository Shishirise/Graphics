## WEBGL Terminology

| Term       | Meaning                                           |
|------------|-------------------------------------------------|
| Vertex     | A point in 3D space                             |
| Fragment   | A pixel shader output                            |
| Shader     | A program that runs on the GPU                  |
| Buffer     | Memory that stores vertices, colors, etc.       |
| Canvas     | The HTML element where WebGL draws the graphics |
